# 게임 업데이트 내용

## 주요 추가 기능

### 1. 몬스터 리스폰 시스템 (5초)
- 플레이어가 몬스터를 밟아 죽이면 5초 후 자동으로 리스폰됩니다
- `game.h`의 `RESPAWN_TIME` 상수로 조절 가능 (현재 100 = 5초)
- 리스폰 시 파티클 효과가 나타납니다
- Enemy 구조체에 `isDead`와 `respawnTimer` 필드 추가

### 2. 스테이지 시스템
- 목표 점수에 도달하면 스테이지 클리어
- **Stage 1**: 500점 목표
- **Stage 2**: 800점 목표
- **Stage 3**: 1200점 목표 (이후 +300씩 증가)
- 스테이지가 올라갈수록:
  - 플랫폼 수 증가 (+2개씩)
  - 적 수 증가 (+1개씩)
  - 코인 수 증가 (+3개씩)
- 다음 스테이지로 넘어가면 체력 +20 보너스

### 3. 게임 재시작 기능
- **R키**: 현재 스테이지 재시작
- **N키**: 스테이지 클리어 후 다음 스테이지로 이동
- 게임 오버 후 메뉴로 복귀하여 재시작 가능

### 4. 메뉴 시스템
- 게임 시작 전 메뉴 화면
- 옵션:
  1. 게임 시작
  2. 랭킹 보기
  3. 종료
- W/S 키로 선택, ENTER로 확인

## 게임 조작법

### 이동
- **A**: 왼쪽 이동
- **D**: 오른쪽 이동
- **W / SPACE**: 점프
- **R**: 게임 재시작
- **N**: 다음 스테이지 (클리어 후)
- **ESC**: 게임 종료

### 게임플레이 팁
1. 몬스터를 위에서 밟으면 50점 획득
2. 몬스터는 5초 후 리스폰되므로 계속 점수를 얻을 수 있습니다
3. 코인 1개당 10점
4. 빨간 몬스터(M): -15 HP
5. 파란 몬스터(W): -25 HP (추적형)
6. 목표 점수를 달성하면 스테이지 클리어!

## 파일 변경 사항

### game.h
- `RESPAWN_TIME` 상수 추가
- Enemy 구조체에 `respawnTimer`, `isDead` 필드 추가
- GameState 구조체에 `stageCleared`, `targetScore` 필드 추가
- `checkStageCleared()`, `nextStage()` 함수 선언 추가

### game.c
- `initGame()`: 스테이지 시스템 초기화
- `initPlatforms()`, `initEnemies()`, `initCoins()`: 레벨에 따라 난이도 증가
- `updateEnemies()`: 몬스터 리스폰 로직 구현
- `checkStageCleared()`: 스테이지 클리어 확인
- `nextStage()`: 다음 스테이지로 이동

### graphics.c
- `drawHUD()`: 스테이지 정보와 목표 점수 표시
- 스테이지 클리어 시 안내 메시지 표시

### main.c
- `showMenu()`: 메뉴 시스템 구현
- `run()`: 게임 재시작 루프 구현
- 스테이지 클리어 후 N키 입력 처리

## 컴파일 및 실행

Visual Studio에서 기존 프로젝트에 파일들을 교체하고 빌드하면 됩니다.

```
빌드 -> 솔루션 빌드
디버그 -> 디버깅 시작 (F5)
```
